Sistema de Inventario

Programa desarrollado en C++ para administrar productos de un inventario.
El sistema permite registrar y mostrar productos guardados en un archivo binario (productos.dat).

Funciones del Sistema
Registrar productos
Mostrar productos almacenados
Guardar información en archivos binarios
Datos del Producto

Cada producto almacena:

Código
Nombre
Categoría
Stock
Precio
Estado (activo o inactivo)
